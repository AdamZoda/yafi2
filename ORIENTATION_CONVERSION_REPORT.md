## 💡 Intégration YAFI

### Code d'Intégration Simple:

```python
# backend/server.py
from pyswip import Prolog

prolog = Prolog()
prolog.consult("knowledge_base_orientation.pl")
prolog.consult("inference_rules.pl")
prolog.consult("yafi_orientation_integration.pl")

def repondre_orientation(question):
    # FAQ d'abord
    for result in prolog.query(f"reponse_faq('{question}', R)"):
        return result['R']
    
    # Puis règles
    for result in prolog.query(f"regle_orientation('{question}', R)"):
        return result['R']
    
    return "Information non disponible"
```

### Utilisation dans YAFI:
```python
@app.route('/api/orientation', methods=['POST'])
def orientation():
    question = request.json.get('question')
    reponse = repondre_orientation(question)
    return jsonify({'response': reponse})
```
